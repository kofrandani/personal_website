Projects

Create Your Own Adventure

Co-developed a text-based adventure game in Python with a collaborator and created a detailed manual and customizable template to help others build similar adventures—from designing narrative structures to deploying the game on Heroku.

The game features branching storylines driven by player choices and tracks hidden decisions to assign scores across 4+1 character traits at the end. The original version spans three chapters, contains over 60,000 words, and offers multiple endings per chapter. The template supports even more ambitious projects.

The project is available here. The manual and template remain publicly available, though the game itself is no longer hosted since Heroku discontinued its free tier on November 28, 2022.

Physarum Polycephalum-Inspired Network Design

Modeled Budapest’s public transport network using slime mould-inspired algorithms based on the adaptive behavior of Physarum polycephalum.

Simulated bio-inspired network growth and compared it to the actual system using graph metrics and targeted disruption tests.

Introduced a novel variation of the Physarum model and analyzed resilience to node- and route-level failures.

Due to confidentiality and ethical agreements, code and data are not publicly shared.

Theses

Master’s Thesis: Causal Impact of Mid-Season Manager Changes

My master’s thesis combined econometric and causal analysis tools to estimate the causal effect of mid-season manager changes in top European football leagues.

The project used web-scraped data and applied a theory-informed causal map with backdoor adjustment, as introduced in Judea Pearl’s Book of Why. The analysis employed First Differences (FD) regression with synthetic control and clustered standard errors, within an event study framework.

The thesis presents a robust, data-driven approach and finds that mid-season managerial changes have a positive, but statistically insignificant effect on end-of-season team standings.

Bachelor’s Thesis: Spatial Analysis of COVID-19 Mortality in Italy

My bachelor’s thesis applies econometric and spatial analysis methods to identify key drivers of COVID-19 mortality during the first wave in Italy (March–April 2020).

Spatial autocorrelation was measured using the k-nearest neighbors method (k=4), complementing the econometric approach in explaining the highly uneven regional death rates across Italy.

Key predictors included the prevalence of multiple chronic illnesses, proportion of women, number of COVID-19 tests, and geographical proximity.
