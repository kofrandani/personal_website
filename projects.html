Create Your Own Adventure

Developed a text-based adventure game in Python with a collaborator and created a detailed manual and customizable template to help others build similar adventures—from designing narrative structures to deploying the game on Heroku. The game features branching storylines driven by player choices and tracks hidden decisions to assign scores across 4+1 character traits at the end. The original version spans three chapters, contains over 60,000 words, and offers multiple endings per chapter. The template supports even more ambitious projects.

The project is available here. The manual and template remain publicly available, though the game itself is no longer hosted, as Heroku discontinued its free tier on November 28, 2022.

Developed a text-based adventure game in python with my collaborator and created a step-by-step manual and template allowing anyone to create their own similar adventure, from designing the page to porting it on Heroku. The game presents multiple choices that define how the story unfolds and tracks the invisible choices the player makes which assign player-specific scores in 4+1 characteristics at the end. The original game has 3 chapters, over 60,000 thousand words and multiple endings per chapter, while the template offers potential to even larger aspirations.

If interested, check out the repository here. The manual and template are still publically avaialble, while the game's link is no longer active since Heroku discontinued its free plans on November 28 2022.
